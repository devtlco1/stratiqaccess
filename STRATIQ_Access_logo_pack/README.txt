STRATIQ Access — Logo Pack

source/    Original uploaded artwork, kept for reference.
svg/       Vector master files (infinite resolution, transparent background).
           - stratiq-icon-*.svg    mark only
           - stratiq-logo-*.svg    full lockup (mark + STRATIQ + ACCESS IRAQ)
           - Color suffix: blue (light backgrounds), white (dark backgrounds), navy (alt)
png/       Raster exports rendered from the vector masters, transparent background.
           - icon/   64 to 2048px square-safe
           - logo/   400 to 2400px wide
favicon/   favicon.ico (16/32/48), apple-touch-icon.png (180, white background),
           icon-192.png, icon-512.png
backgrounds/  Flattened logo-on-white and logo-on-navy, PNG + JPG, for docs/email/slides.

All vector and PNG files were traced from the original artwork with potrace, so edges
are clean vector paths rather than upscaled/blurred pixels.
